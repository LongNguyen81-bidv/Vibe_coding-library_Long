Chỉ sử dụng khi mới clone project này về để bắt đầu làm việc với 1 project mới. Nếu project đã có những file, thư mục khác ngoài các thư mục commands và rules trong .cursor và file README.md thì cảnh báo người dùng không sử dụng lệnh này.
Quy trình:
Xóa thông tin về git của project cũ.
Khởi tạo git cho project mới.
Thêm các file và thư mục hiện tại vào project mới.
Commit thay đổi.
Push lên remote (cần đường dẫn của remote, nếu chưa có remote thì hỏi lại người dùng, tên branch mặc định là main nếu người dùng không chỉ định).
